# HEALTH SCAN PRO

HEALTH SCAN PRO is a web application that uses Artificial Intelligence to analyze CT scan images and help detect certain diseases. The project provides a simple interface for uploading or scanning medical images and viewing AI prediction results.

## Features

- Detect COVID-19 from lung CT scans.
- Detect brain diseases from brain CT scans.
- Detect kidney diseases from kidney CT scans.
- Upload CT scan images.
- Scan images using the device camera.
- Simple and responsive user interface.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- TensorFlow.js
- Teachable Machine Image

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/health-scan-pro.git
```

2. Open the project folder.

3. Run the project using Live Server or any local web server.

4. Open `index.html` in your browser.

## How to Use

1. Open the website.
2. Choose the disease you want to detect.
3. Upload a CT scan image or use the camera.
4. Wait for the AI prediction.
5. View the result.
